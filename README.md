## ⚙️ Instalação
```
# Abra um terminal e copie este repositório com o comando
$ git clone https://github.com/GBDev13/dashgo.git
```

```
# Acesse a pasta da aplicação
$ cd worldtrip

# Instale as dependências
$ yarn or npm

# Inicie a aplicação
$ yarn start or npm install

```
